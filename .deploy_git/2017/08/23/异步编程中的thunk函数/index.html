我们接下来自己实现一个简易的thunk工具方法，思路如下：

  1. 一共三个执行函数，第一层函数传入主函数方法fn，并返回一个带有fn方法的闭包
  2. 第二层函数传入参数args，然后返回一个仅需传入回调函数供外部执行的方法
  3. 第三层函数传入一个回调函数cb, 然后执行主程序
